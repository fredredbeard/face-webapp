# Client-Side Face Detection Application

A real-time, privacy-focused face detection web application that runs entirely in the browser. Powered by TensorFlow.js and optimized with a WebAssembly (WASM) backend for high-performance tracking on consumer hardware.

To run the Web Application, open the link:

# How to Run Locally

https://fredredbeard.github.io/face-webapp/

Tested Environments
This application has been validated on the following configurations:

Mac OS Sequoia 15.6 (M1 Chip):

  - Chrome v143, Firefox v146, Safari v18.6 (Smooth execution via WASM).

Windows 11:

  - Chrome v143 (Stable >30 FPS).

Samsung Galaxy S25 Ultra (Android 16):

  - Chrome Mobile v143 (Portrait mode, responsive).

iPhone 13 (iOS 26.1):

  - Safari Mobile v26.1.


No Data Collection: the app does not store, record, or transmit your video feed.
